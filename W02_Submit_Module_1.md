## Accomplishments

I completed a console-based task manager using JavaScript and Node.js. The program allows the user to add, list, update, delete, complete, and search tasks. Task information is saved in a JSON file so the data persists between runs.

The project demonstrates JavaScript functions, variables, conditionals, loops, arrays, objects, JSON file storage, and error handling. It also uses ES6 array methods including `map`, `filter`, `reduce`, `find`, `findIndex`, and `flatMap`.

## Unique Module Requirements

* Display output to the terminal: completed through the command-line interface.
* Native Array ES6 functions: completed in task listing, searching, statistics, and id generation.
* Recursion: completed in the nested subtask minute calculation and nested subtask rendering.
* External JavaScript library: completed by using the `picocolors` npm package for terminal styling.
* Stretch challenge: completed by throwing and handling custom validation exceptions.
